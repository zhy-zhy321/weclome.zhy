# 🖥️ VS Code + Git 测试

## 今天学到的 VS Code 操作

| 操作 | 图形界面 | 等价命令行 |
|------|---------|-----------|
| 暂存文件 | 点击 + 号 | git add |
| 提交 | 点击 ✓ | git commit |
| 推送 | 点击 ↑ | git push |
| 查看修改 | 点击文件名 | git diff |

## 感受
用图形界面操作 Git 比命令行直观多了！
不用记那么多命令，点点鼠标就行 😊

## 但是！
命令行依然重要，因为：
- 服务器上没有图形界面
- 有些操作图形界面做不到
- 面试可能会考命令行

两个都要会！
