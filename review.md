# 🧠 Git 命令复习 — Day 1

## 终端基础命令
| 命令 | 作用 | 我的理解 |
|------|------|---------|
| pwd | 显示当前路径 | 我在哪？ |
| ls | 列出文件 | 这里有什么？ |
| cd 目录名 | 进入目录 | 我要去那里 |
| cd .. | 返回上一层 | 退一步 |
| mkdir | 创建文件夹 | 建新房间 |
| clear | 清屏 | 打扫屏幕 |

## Git 基础命令
| 命令 | 作用 | 我的理解 |
|------|------|---------|
| git init | 初始化仓库 | 给文件夹装上Git |
| git clone URL | 克隆仓库 | 下载项目 |
| git status | 查看状态 | 随时看看情况 |
| git add . | 添加到暂存区 | 放进购物车 |
| git commit -m "信息" | 提交 | 结账保存 |
| git push | 推送到远程 | 上传到GitHub |
| git pull | 拉取最新 | 下载最新代码 |
| git log --oneline | 查看历史 | 翻看存档记录 |
| git diff | 查看差异 | 对比改了什么 |

## Git 分支命令
| 命令 | 作用 | 我的理解 |
|------|------|---------|
| git branch | 查看分支 | 有哪些平行宇宙 |
| git switch -c 名字 | 创建并切换分支 | 开一个新宇宙 |
| git switch 名字 | 切换分支 | 跳到另一个宇宙 |
| git merge 名字 | 合并分支 | 两个宇宙合体 |
| git branch -d 名字 | 删除分支 | 关掉旧宇宙 |
